PlugY does store inventory data in local files but so does Diablo 2 (via 
PvPGN on the server at least). The single player <--> multi-player 
shared stash for PlugY would have to be disabled because that would be a 
bad idea for any server. But the multi-page stash should be possible via 
PlugY if the server allowed it to run and PvPGN handled the file saving 
server side fine. Currently PlugY is automatically disabled for closed 
Bnet games but older versions didn't and the problem was that D2GS 
detected PlugY as a cheat and wouldn't allow it. There would probably be 
some bandwidth limitations depending on how many extra pages of stash 
were allowed and a limit would probably have to be enforced. I know game 
load times increase in open games when there are a lot of stash pages 
filled so larger servers would probably set lower limits to reduce 
bandwidth usage.
