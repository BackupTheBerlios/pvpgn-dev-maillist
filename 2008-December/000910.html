Hi d1zzy,

What do you expect to get out of this "after all is said and done" ?

The idea of a secure "tunnel" for these cd-key checks makes sense no
matter how you look at it. The fact that Blizzard didn't go for this
option was their own choice.

With PvPGN attempting to intersect/proxy these requests I think it will
restart Blizzard's hunt to close PvPGN down. I think, if PvPGN is to do
cd-keys, it should be via the secure tunnel, and every admin is giving
a server login.

If anything is tracked to that server admin, they will know where to
look (stolen keys etc.)

Either way, what would happen if the key failed a check ? What good 
is it to PvPGN to check the key (not talking legally now :) )

Cheers
