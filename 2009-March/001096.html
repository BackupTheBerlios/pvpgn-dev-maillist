Hi,
this command is quite similar to the lockacct command.

Mute remove the ability from a user to chat on any channel on the server or whisper to other users.

The user can still talk inside games of ladder/custom (he still won't be able to whisper even if he is inside a game)

It's a great add-on, and if it's possible I would like to be added to the pvpgn source code permanently 

Thx a lot, Shakar from EuroBattle.net.

PS: for those who want to try it, don't forget to add the commands to the command_group.conf
