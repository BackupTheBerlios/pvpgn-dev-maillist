The old loader still works just fine, I just extended the code to also 
handle the original WAR3 authentication scheme.

The (current) status:
  - "old" accounts can't login with SRP3
  - accounts created via SRP3 can't login via SC hash
     (creates bogus more or less random SC hash)
  - accounts newly created with old loader will be
     able to login via SRP3 and via SC hash
  - the /passwd command does not yet create SRP3 credentials
  - no clue yet if it possible to extend account creation via
     other blizzard clients to create SRP3 credentials
  - SRP3 passchange not yet implemented (work in progress)
  - code not yet tested on anything but Linux/AMD64
  - some endianess optimizations might still be possible
     (I think I', swapping endianess twice - maybe I can do
      without)
  - MacOS without server signature check should at least be
     able to login - not sure if ingame works

Hope this answers all important questions so far.
