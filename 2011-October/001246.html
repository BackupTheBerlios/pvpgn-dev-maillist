For git I'm using Gitorious - it is outside USA in Norwegian but it does not 
support bug tracker, mailinglist and website hosting. And Gitorious removed 
PS3 jailbreak too (DMCA).

Next, I'm using Launchpad, it is very good project hosting server outside USA 
(whois telling me it is registred in GB), but it does not support git - only 
bzr and it does not support website hosting (all other it has: mailinglist, 
bugtracker, support/answer tracker, multi bzr code repositories with merge 
support, blueprints, source code autobuilder for Ubuntu).

I think that all hosting servers are on wiki: 
http://en.wikipedia.org/wiki/Comparison_of_open_source_software_hosting_facilities

Next option could be to use some non-US free web hosting service where can be 
bugzilla and dvcs system installed + use external mailinglist.
