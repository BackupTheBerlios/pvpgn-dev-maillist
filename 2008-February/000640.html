Comment:
/clanadd -> add clan
/clandel -> remove clan
/clanaddu -> add user to clan
/clandelu -> remove user from clan
/delacct ->  delete account from hash ( use it before delacctsql )
/delacctsql -> deletes acct from sql
/flushacct -> load account and account's friend list
-----------
account renaming sql conf modify:
"acct_username varchar(32)","NULL" 
"username varchar(32)","NULL"
//
modify with set command the username and acct_username of the user
ex /set xy BNET_username zx
user /delacct to delete it from hash table
use /flushacct to reload the renamed account
