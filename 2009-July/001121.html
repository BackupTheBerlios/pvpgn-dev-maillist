Summary: PvPGN can only send 19 active custom games for a list update

Details: Basically, the server is unable to send more than a certain number of bytes per a 'packet', bnetd appears to limit each game list refresh to a single 'packet', which limits the number of games that can be sent at a time.

If there are more than 19 active games, games become 'invisible' (the server does not send information about them) on the list. The game object does still however exist and users are able to connect to it, if they happen to know the game name.
