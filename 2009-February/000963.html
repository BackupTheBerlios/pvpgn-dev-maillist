Comment:
This command especially designed for the W3 custom games. normal
/games command lists all the avaible games but in a big pvpgn server
its not readeble. So implemented the new /filter_games command.
Example;
/filter_games DOTA ; will filter out the games with 'DOTA' in it's
name and return to user.
Note: It do not uses wildcards but uses the simple strstr method for
finding filter string in the game name.

Altough I tested it w3 but needs more testing..

Note2: I'm looking for a way to improve /games and /filter_games
ouput,ie make it more readeble. I'm open for suggestions.
