Hi JEBs,

and thanks alot for your valuable feedback.
We are currently preparing yet another 1.8 branch release,
so its CVS we need tested this time.

About the missing sha1pass.cpp: It was just a test app to see if
my implementation does work as intended or not. I first didn't
even intend to include it anyways... but I just decided that we
can include it in SVN (now named sha1hash) as it makes it easier
to test the sha1 implementation on other architectures, too.

So anyone on a 64bit machine (or a machine with byte order differing
from x86) is welcome to test it and report back if the results are
wrong.

test = a94a8fe5ccb19ba61c4c0873d391e987982fbbd3
TEST = 984816fd329622876e14907634264e6f332e9fb3

(I haven't tested 'long' messages yet (especially >53 and >64 bytes),
so those might be worth an extra test, too...)
