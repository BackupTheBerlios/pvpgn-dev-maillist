First, thx all for your hard work.

All4you, try like this:
1. cd /usr/local/src
2. svn checkout http://svn.berlios.de/svnroot/repos/pvpgn/trunk trunk
3. cd trunk/pvpgn/
4. ./autogen.sh
5../configure
6. make
7. make install

2. svn checkout --revision 282 http://svn.berlios.de/svnroot/repos/pvpgn/trunk trunk

I'm sure pelish will fix this soon.
After that you do an update like this:
1. svn update http://svn.berlios.de/svnroot/repos/pvpgn/trunk trunk
2. make
3. make install
