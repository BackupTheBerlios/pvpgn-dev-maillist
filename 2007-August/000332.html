Diablo high-level MOD patch, which enables the storage of ladder entries > lvl 255.
In addition it clears up some old and wrong level sizes (level (in d2cs/d2dbs) should be bn_short instead of bn_byte level + bn_byte u1).
The in-game ladder list is limited to the 0xff cap but shows the entries now in the right order instead of scrambled.

CRITICAL patch, because it changes the internal Diablo ladder file structure and therefore makes old ladderfiles incompatible.

Patch might be better remain external.
