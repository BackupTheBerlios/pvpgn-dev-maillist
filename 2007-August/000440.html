Hi

We are trying out a cmake build system for pvpgn that in theory should
be more portable (should allow us to use the same build system for all
pvpgn supported platforms and not have to maintain yet another build
system for Windows and others).

First we need to establish it works as weel as autotools works. This
means everyone has to test it out on all platforms he has access to
and where he knew pvpgn compiled fine with autotools.

To get yourself acquainted with using cmake to build pvpgn you should
check out README.cmake (unless you already know cmake well). If you
find out problems with cmake where autotools worked please get back to
us, with platform information and what did you tried and what doesn't
work.
