This "bug" seems to me trivial. The issue goes when you have allowed WOL support in 1.99.x so it means that you open ports 4000/4005/(4807/5400). D2 server use also port 4000 so you can:
1) change bnetd WOL port 4000 to someone else. That will in fact disabled WOLv1 game clients (wich are Command n Conquer/Red Alert1 v2) but yoiu will be possible to have D2 server and WOL server on the SAME IP Adress.
2) use an other IP for D2 server. You will have support for all clients that is possible.

Im recommending the second option.
