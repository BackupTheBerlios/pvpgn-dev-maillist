need to add those lines to bnetd.conf:

# should users activate their accounts from time to time? 
# setting this value "true" will deney access to the server unless the account is activated 
# doesn't work on chat 
enable_activation = true

this patch basically, force the users to come once a while to the web site. and activate their account.

our activation site is: http://app.eurobattle.net/activate
you can't put this on true unless you got a site who does that.

this patch was created by madmax, and I added the option disable/enable in bnetd.conf few days ago.

madmax added a new table to the database called Custom (my guess is that you will want to use BNET)

also, maybe add a variable instead of just writing "Activate account at www.eurobattle.net"
