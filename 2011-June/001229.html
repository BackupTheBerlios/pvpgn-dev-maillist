Hello,

I tried use bnchat/bnstat programs to connect to eurobattle.net pvpgn
server, but these programs cannot connect.

When I configured my own pvpgn server which accept all clients with
all version, all worked fine. When I enable only one specified war3
version, bnchat/bnstat cannot connect to my server too (like on
eurobattle.net). I found problem and I'm sending patch which fix it.

This patch add option -i / --ignore-version in client_connect.cpp
which does not send game version information, cd owner and cd key
after SERVER_AUTHREQ_109 (like program bnbot). When I used bnchat with
--ignore-version I was able connect without problems to my server and
server.eurobattle.net too.
